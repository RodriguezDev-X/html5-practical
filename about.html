Short History

Origins of HTML

HTML was first proposed by Tim Berners-Lee in 1991.

  • HTML 1.0 — Basic structure
  • HTML 4.0 — Widely adopted
  • HTML5 — Modern, semantic, multimedia-friendly

Milestones

  1. 1991 — First website published
  2. 1995 — HTML 2.0 released
  3. 2014 — HTML5 becomes W3C recommendation

Definitions

HTML
HyperText Markup Language, the standard for web pages.
W3C
World Wide Web Consortium, the body that standardizes web technologies.
Semantics
Using meaningful tags that describe their purpose.
